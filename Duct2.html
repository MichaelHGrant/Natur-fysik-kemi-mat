Interactive webpage version of the HP BASIC duct acoustics code

This page translates the visible HP BASIC “Material” and “Extended” routines into modern JavaScript. It uses the Delany–Bazley / Mechel material model and solves the extended-reaction equation f(w) = w tan(w) + M √(w² + p²) tan(b √(w² + p²)) = 0 across frequency using Newton–Raphson with Muller fallback.

It is designed as a faithful interactive webpage version of the code pages you uploaded, not as a perfect forensic transcription of every line. The visible code pages define the key structure, constants, and solver flow for “Material” and “Extended”.
