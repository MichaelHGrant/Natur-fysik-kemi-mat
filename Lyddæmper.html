Fysikken og Matematikken bag Q4POLE

Dette program er baseret på Overførselsmatrix-metoden (Transfer Matrix Method), også kendt som "Four-Pole Parameters" i akustikken. Det er en elegant måde at beregne, hvordan lyd udbreder sig gennem et system af forbundne rør, som f.eks. en biludstødning eller et ventilationssystem.

1. Lydbølger som et kredsløb

I stedet for at løse komplekse 3D-bølgeligninger, antager vi, at lyden udbreder sig som plane bølger (1D). Vi kan beskrive tilstanden i ethvert snit af røret med to variabler:

  • p: Lydtrykket (svarer til spænding i elektronik)
  • v: Volumenhastigheden af luften (svarer til strømstyrke i elektronik)

2. Fire-pols Matricen (The Four-Pole Matrix)

For et lige rør med længden L, tværsnitsarealet A og lydens bølgetal k = 2πf/c, kan forholdet mellem indgangen og udgangen beskrives med en 2x2 matrix:

[ p1 ] = [ A   B ] * [ p2 ]
[ v1 ] = [ C   D ] * [ v2 ]

Hvor de fire "poler" er komplekse tal givet ved Euler's relationer for bølger:

  • A = D = cos(kL)
  • B = j · Z · sin(kL)
  • C = (j / Z) · sin(kL)

Hvor Z er den akustiske impedans (Z = ρ · c / A), ρ er luftens densitet, c er lydens hastighed, og j er den imaginære enhed.

3. Sammensætning af systemet

Når lyden bevæger sig gennem flere sektioner, kan vi finde det samlede systems egenskaber ved simpelthen at multiplicere matricerne sammen:

Mtotal = Msektion_1 × Msektion_2 × Msektion_3 ...

4. Transmissionstab (TL)

Til sidst udregnes Transmissionstabet i decibel (dB). Det udregnes fra de endelige A, B, C og D parametre i den samlede matrix Mtotal, korrigeret for de forskellige rørdiametre ved selve indgangen (kilden) og udgangen (luften udenfor).
